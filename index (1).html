Portfolio — tampilkan file asli (PPTX)

Halaman ini menampilkan file PPTX original yang tidak diubah sama sekali. Pastikan file PPTX (namanya Portfolio_DINI LUSPITA-1.pptx) ikut di-upload ke folder yang sama saat kamu deploy.

1) Unduh file PPTX asli

Klik tombol di bawah untuk mendownload file presentasi aslinya (tidak ada perubahan).

Download PPTX asli

Jika file bernama berbeda, ganti nama file di server jadi Portfolio_DINI LUSPITA-1.pptx atau sesuaikan link download.

2) Tampilkan langsung (browser behavior bervariasi)

Beberapa browser tidak langsung merender file PPTX dan akan mengunduhnya. Namun, kami menyertakan iframe/object yang mencoba menampilkan file apabila browser mendukung preview.

Jika iframe di atas tidak tampil (hanya download), itu karena browser tidak mendukung preview PPTX. Lanjut ke opsi nomor 3 untuk pengalaman yang paling mirip tampilan asli.

3) (Direkomendasikan) Tampilkan via Microsoft Office Online Viewer

Office Online Viewer dapat menampilkan PPTX persis seperti aslinya dalam browser. Caranya: upload file PPTX ke GitHub (atau server publik), lalu gunakan URL raw file di Office Viewer.

Langkah singkat

  1. Upload file Portfolio_DINI LUSPITA-1.pptx ke repository GitHub kamu di folder root.
  2. Dapatkan link raw file GitHub:
    https://raw.githubusercontent.com/USERNAME/REPO/main/Portfolio_DINI%20LUSPITA-1.pptx
    -- Ganti USERNAME dan REPO sesuai akun & repo kamu. Perhatikan spasi diganti dengan %20 pada URL raw jika ada.
  3. Buka Office Online embed URL berikut (copy-paste di browser):
https://view.officeapps.live.com/op/embed.aspx?src=PASTE_RAW_FILE_URL_HERE

Contoh lengkap (ganti USERNAME & REPO):

https://view.officeapps.live.com/op/embed.aspx?src=https://raw.githubusercontent.com/USERNAME/Portfolio-Dini-Luspita/main/Portfolio_DINI%20LUSPITA-1.pptx

Jika ingin, kamu bisa menaruh iframe Office Viewer di file index.html setelah mengganti PASTE_RAW_FILE_URL_HERE:

<iframe class="viewer" src="https://view.officeapps.live.com/op/embed.aspx?src=PASTE_RAW_FILE_URL_HERE"></iframe>

Dengan cara ini, tampilan slide akan sama persis seperti file aslinya.

Catatan teknis penting

  • Jangan ubah file PPTX asli — halaman ini hanya menunjuk ke file aslinya.
  • Pastikan file PPTX di-upload ke folder yang sama saat kamu deploy (root repository) agar link relatif bekerja: Portfolio_DINI LUSPITA-1.pptx.
  • Jika repo GitHub mengandung spasi di nama file, raw URL akan meng-encode spasi menjadi %20.
Buat oleh asisten — file asli tetap utuh dan tidak diubah.
